📚 BTS SIO Option SLAM
Le BTS SIO a été mis en place en 2011 pour remplacer l’ancien BTS Informatique de Gestion

Cette formation se divise en deux options SISR et SLAM

L’option SLAM est celle qui se concentre sur le développement informatique

Elle forme à la création d’applications de sites web et logiciels adaptés aux besoins d’une organisation
